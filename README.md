## 🌟 Overview

A clean, animated, and fully responsive portfolio website showcasing my skills, projects, certifications, and contact information. Built with a focus on performance, accessibility, and user experience — featuring dark/light mode, smooth scroll animations, and an interactive UI.

---

## ✨ Features

- **Dark / Light Mode** — System-aware theme toggle with smooth transitions
- **Animated UI** — Fluid animations powered by Framer Motion
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Smooth Scrolling** — Section-based navigation with `react-scroll`
- **Contact Form** — Functional email integration via EmailJS
- **Loading Screen** — Custom animated loading experience
- **SEO Ready** — Semantic HTML with proper meta structure
- **Fast Build** — Powered by Vite with TypeScript support

## 🚀 Getting Started

### Prerequisites

- **Node.js** v16 or higher
- **npm** or **yarn**

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/cjhimanshu/cj_portfolio.git
cd cj_portfolio

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
